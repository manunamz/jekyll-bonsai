### Navigate

To navigate to the archive page, just hover over 🗺 and click on ⛲.

### Markdown

```markdown
---
layout: archive
permalink: /archive/
title: <your-title-here>
---

<your-text-here>
```

### Config

Defaults are listed below:

```yaml
bonsai:
  archive:
    section:
      posts: "🫐 🥭 🍈 Fruits"
      books: "🌰 🍵 🎍 Germs"
      entries: "🌱 🌿 🎋 🌸 Foliage"
```

`section`: Displays the section subtitles for the given collection type (`posts`, `books`, `entries`).
